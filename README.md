# 阳的数据大屏 (YangBigScreen)

一个从 0 到 1 教你如何动手制作数据可视化大屏的开源项目。

## 项目简介

本项目旨在帮助开发者从零开始，一步步学习和掌握数据可视化大屏的开发流程。无论你是前端新手还是有一定经验的开发者，都可以通过本项目掌握数据大屏的核心技术栈和实现思路。

## 技术栈

- Python
- 前端可视化技术（持续更新中...）
